## Run Project
### Clone locally
```
git clone https://github.com/ayoubmehd/ayoub.elmahdaoui-taas-frontend-challenge.git
```
### Install dependencies
#### server
```
yarn
```
### client
```
cd client
yarn
```
### Compiles and hot-reloads for development
```
yarn dev
```
